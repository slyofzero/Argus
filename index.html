world engine

CREATING THE WORLD ENGINE

  • Sharded Rollup SDK

    World Engine’s novel layer-2 sharding architecture empowers game developers to build their own horizontally scalable blockchain that inherits the security of the underlying layer-1 blockchain.

  • Made for Games, Not JPEGs

    Inspired by the server architecture behind the most computationally intensive online games, World Engine's game shards allow developers to build their games fully onchain.

  • EVM-compatible

    Build your rollup on top of the largest blockchain developer ecosystem: Ethereum. The World Engine-based chain is EVM-compatible and supports all Ethereum wallets and toolings.

work pillars

FOUNDATIONAL PILLARS OF ARGUS

  • 001 Research Lab

    We build novel infrastructure, primitives, and game economy design to advance the World Engine ecosystem and crypto gaming as a whole.

  • 002 Game Studio

    Argus is a games company, not a pick-and-shovel company. We build games on the World Engine stack and trailblaze the path to realizing the Internet of Games.

  • 003 Game Publisher

    We help game studios building on the World Engine go from 0 to 1 through our edge in crypto gaming infrastructure, research, and growth.
